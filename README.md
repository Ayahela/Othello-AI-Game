# Othello Game with AI (Minimax + Alpha-Beta Pruning)

This project implements the classic board game **Othello (Reversi)** where a human player competes against an AI agent.

The AI uses **Minimax algorithm with Alpha-Beta pruning** to make optimal decisions.

---

## 🧠 AI Techniques Used

- Minimax Algorithm
- Alpha-Beta Pruning
- Depth-limited Search (depth = 3)
- Heuristic Evaluation Function (disc difference)

The AI evaluates board states and chooses the move that maximizes its advantage while minimizing the opponent's.

---

## 🎮 Features

- Full Othello game logic
- Human vs AI gameplay
- Graphical User Interface using Tkinter
- Valid move highlighting
- Score tracking
- Automatic game-over detection
- Pass-turn handling

---

## 🛠 Technologies Used

- Python
- Tkinter (GUI)
- Object-Oriented Programming
- Game Tree Search
